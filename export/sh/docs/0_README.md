## 🆘 获取帮助

### 脚本帮助

```bash
# 中文帮助
./docker-export-compose.sh --help-cn

# 英文帮助
./docker-export-compose.sh --help

# 版本信息
./docker-export-compose.sh --version
```

### 快速命令

```bash
# 基础导出
./docker-export-compose.sh my-container

# 生产环境（推荐）
./docker-export-compose.sh --type env --privacy my-container

# 查看完整文档列表
ls docs/
```
